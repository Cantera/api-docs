15//! Factory class to create Func1 objects
16//!
17//! This class is mainly used via the newFunc1() function, for example:
18//!
19//! ```cpp

21//! ```
22//! @since New in %Cantera 3.0

25public:
26 /**

32
33 void deleteFactory() override;

35private:
