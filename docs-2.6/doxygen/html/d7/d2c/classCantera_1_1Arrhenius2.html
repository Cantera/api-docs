Detailed Description

Arrhenius reaction rate type depends only on temperature.

A reaction rate coefficient of the following form.

\[ k_f = A T^b \exp (-E/RT) \]

Deprecated:
To be removed after Cantera 2.6. See ArrheniusRate.
