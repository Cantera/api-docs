Detailed Description

Advanced functors implement expressions that require multiple parameters.

The following advanced functor types are implemented:

  • "tabulated-linear" and "tabulated-previous" (class Tabulated1),
  • "polynomial3" (class Poly13),
  • "Fourier" (class Fourier1),
  • "Gaussian" (class Gaussian1),
  • "Arrhenius" (class Arrhenius1).
